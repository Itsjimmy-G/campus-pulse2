# 🏥 Campus Pulse | Nite Studio
**Agile Event Management System**

## 🚀 Easy Setup for Colleagues
1. **Clone the repo.**
2. **Create environment:** `python -m venv .venv`
3. **Activate:** `.\.venv\Scripts\Activate.ps1`
4. **Install everything:** `pip install -r requirements.txt`
5. **Run:** `python run.py`
